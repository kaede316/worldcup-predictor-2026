# 2026 FIFA World Cup Predictor

2026 美加墨世界杯 AI 预测工具。覆盖小组赛全赛程，集成四维数据验证、泊松比分预测、全球 LLM 共识面板。

## 特性

- 完整小组赛赛程（6 月 12-24 日，I、J、K、L 组第 1-2 轮）
- 实时赛果 + AI 预测比分
- 胜平负概率条
- 关键因素标签（组别、实力对比、场地特征）
- 迷你日历 + 悬浮赛程浮层
- 全球 8 大 LLM 模型预测共识排行榜（HF Leaderboard）
- 深色主题 · 响应式适配

## 使用

直接在浏览器打开 `index.html`。

## 数据来源

赛程基于 FIFA 官方公布信息，预测由 AI 模型生成，仅供参考。
